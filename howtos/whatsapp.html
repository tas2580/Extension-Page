Get your WhatsApp® password

If you use my Mobile Notifier Extension you need a phone number as username and a password to login into the WhatsApp® account that sould send the messages. For that you need a phone on that you can recive SMS or calls because you need to register a new WhatsApp® account on the phone number. WhatsApp® can only be used once per number, so you need a phone number on that you use no WhatsApp®.

On Windows

If you use Windows on your computer, you can use WART to get your WhatsApp® password.

Run WART and enter your phone number with country code and without 0 at the beginning. Choose whether you want to receive a call or want to get a SMS. Wait until you have received the Code and enter the Code to WART thereupon WART querys your password from WhatsApp® and showes it to you.

On Linux

As a Linux user you can use yowsup to get your password from WhatsApp®. Open a command line window and type:

wget https://github.com/tgalal/yowsup/archive/master.zip
unzip master.zip
cd yowsup-master/

On Debian you need the python-dateutil package. Install it with apt-get install python-dateutil

After that edit the config file called config.cfg and enter the following to it

cc={CC}
phone={CC}{PHONE NUMBER} (without 0 at the beginning)
id=
password=

Replace {CC} with your country code and {PHONE NUMBER} with your phone number.

Now you can run yowsup to get a SMS with a Code. If you whant to recive a call replace sms with call in the command.

python yowsup-cli registration -c config.cfg -r sms

After you have recived the Code run the following command, replace {YOUR CODE} with the Code you have recived. You need only the numbers without the hyphen.

python yowsup-cli registration -c config.cfg -R {YOUR CODE}

Now yowsup will display you the password and some other data about your WhatsApp® Account.
