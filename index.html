Welcome to my phpBB Extensions page

On this website you can find information about the phpBB extensions that are developed by tas2580. If you need further extensions, look at the Extensions database on phpbb.com.

What are phpBB Extensions?

Extensions are downloadable packages which can be used to modify a phpBB forum or add new functions to the forum. Extensions can be used since phpBB 3.1. As opposed to Modifications in phpBB 3.0 there are no code changes needed for the installation of Extensions. To install an Extension, it must only be uploaded and activated in the ACP, that makes customising phpBB easier than ever.

The most phpBB Extensions are writen and suported by the community itself, there are only a few offical Extensions that are developed and supported by the phpBB team.
